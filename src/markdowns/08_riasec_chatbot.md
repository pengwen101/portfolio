# RIASEC Assessment and Job Recommendation System

![alt text](/img/riasec_chatbot_1.png)
![alt text](/img/riasec_chatbot_2.png)

A chatbot utilizing the LlamaIndex library and llama LLM model to give career recommendations that is connected to Petra Alumni API based on the user's RIASEC test result. Users will take an official RIASEC test first, then the data will be fed into LLM ReAct agent to generate keywords and fed it into the API to give real job vacancy offers.

My Contributions:
- Utilized LlamaIndex ChatEngine to give career recommendation to users based on user’s RIASEC test result
- Implemented multiagent concept using ReAct agent and tools to search from Petra Alumni Website based on user’s RIASEC test result
- Implemented swarm multiagent to give educational content to users based on user’s RIASEC test result
