# Search Jobs & Analyze Skill Gaps Based on RIASEC Traits

![alt text](/img/riasec_gap_analysis.png)

A web application utilizing multi-agent LLM to recommend job vacancies sourced from Petra's alumni website and analyze the gaps between a user's RIASEC traits and a job's description or requirements. It further suggests educational videos to help bridge those gaps.
