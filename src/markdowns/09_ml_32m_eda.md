# MovieLens-32M: PySpark Exploratory Data Analysis

![alt text](/img/ml_32m_eda.png)

This project contains a PySpark notebook designed to process and analyze the MovieLens 32M dataset. The pipeline reads the raw tables (ratings and movies) directly from Databricks Unity Catalog and performs distributed data transformations to prepare the massive dataset for Exploratory Data Analysis (EDA).
