# Content-Based Recommendation System For Careers, University & Majors, Courses & Certifications, and Job Vacancies

![alt text](/img/content_based_recsys.png)

In this project, my partner and I scraped data on job vacancies, online courses, and university programs. We implemented a hybrid approach using the O*NET API and text similarity to build a system that recommends jobs and courses based on a user's RIASEC score, skills, interests, and academic major. This tool is designed to help high school and university students select the right career path; for high school students, it also recommends suitable university programs. We built the backend using FastAPI and utilized FAISS for vector storage. This work was part of a larger collaborative effort involving frontend, marketing, and project management teams for the final project of an ML Ops workshop hosted by RISTEK Universitas Indonesia.
